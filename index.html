火場無線電通訊關鍵語詞研究

專家訪談表版型設計（封面＋內頁）
本研究旨在分析火場無線電通訊語句，建立安全導向之關鍵語詞分類，作為語音辨識（ASR）與自然語言處理（NLP）模型之基礎。版型設計以「正式、清晰、可讀、易填寫」為原則，適合紙本訪談、PDF 匯出與 HTML 部署展示。

研究核心模組

WWWS 通訊結構 Who / Where / What / State 安全語詞辨識 Mayday 判斷
研究定位 聚焦安全語詞，而非以戰術操作細節為核心。
資料特性 以火場實務經驗與專家判斷，建立語詞使用與重要性依據。
用途說明 作為 ASR / NLP 模型訓練與安全語意分析之基礎資料。

語意分類重點

危險警示 例如煙色變化、異常聲音、火勢轉強、環境徵象。
人員狀態 例如氣量不足、迷失方向、撤離中、裝備異常。
緊急求救 例如 MAYDAY、未喊 MAYDAY 但已屬危急之情境表述。

HTML 部署建議架構

Deployment Structure
建議將三份 checklist 不要拆成三個完全分離的 HTML，而是整合成同一個系統頁面，以上方分類切換或頁內區塊方式呈現。這樣可維持相同設計語言、方便專家連續填答，也利於後續匯出 JSON 或整合研究資料。
部署層級 建議做法 用途
